Retro Custom Markup Demo Page (using prototype)


Setup

Set up is easy. First include prototype.js: 
<script src="jquery.js" language="javascript" type="text/javascript"></script>
Then the retro engine: 
<script src="retro.js" language="javascript" type="text/javascript"></script>
Then include whatever markup languages you have. This demo page includes the following: 

<script src="prototype_demo_ml.js" language="javascript" type="text/javascript"></script>
